
# **03_TOPページのHTML（index.html）**

## **TOPページのHTMLを書いてみよう**

- [ ] サンプルプログラムのダウンロード
- `[サンプルプログラム]>sample.zip`
- `[プログラミング]>`の中に展開
<br><br>

- [ ] ファイル・フォルダ作成(復習) 
- `[プログラミング]`の中に`[cat]`フォルダ作成
- `[プログラミング]>[cat]`の中に`[images]`フォルダ作成(画像を入れる場所)
- `[プログラミング]>[cat]`の中に>`index.html`ファイル作成
